## Installation Guide

ENV Parameter must be change.
APP_FORCE_TLS=false
APP_ALLOW_INSECURE_HOSTS=true
APP_TIMEZONE='UTC'

1. Create database first
2. Setup ENV for Database section
3. Run App url
4. Create Database,
5. Entry your outlet & company profile with user access
6. enjoy
