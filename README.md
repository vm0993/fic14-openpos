## Installation Guide

ENV Parameter must be change.
1. APP_FORCE_TLS=false
2. APP_ALLOW_INSECURE_HOSTS=false
3. APP_TIMEZONE='UTC'

Step by step

1. Create database first
2. Setup ENV for Database section
3. Run App url
4. Create Database,
5. Entry your outlet & company profile with user access
6. enjoy
